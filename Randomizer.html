About the Randomizer

Introduction

The Transformers 2004 Randomizer is now a tool built in to the TF04 File Converter. As of TF04 Randomizer's version 1.0, the Randomizer will relocate all Minicons and optionally all Datacons. These collectables can be placed in any level at any location that existed in the original game (with a couple custome locations thrown in). Using user-customizable logic, you can adjust the difficulty of the Randomizer to meet your desired challenge (details on this below). For convenience, the TF04 File Converter also provides tools to unpack and repack game ISOs to make setting up the Randomizer a simple process. The Randomizer also provides an opportunity to incorporate some cut content back into the game, bringing a little extra surprise to every run.

Requirements

To use the Randomizer, you first need a legally acquired copy of the Transformers (2004) ISO. You will also need both 7Zip and ImgBurn installed on your computer. You can use the programs manually (not recommended, it's a pain), or you can set the EXE locations in the TF04 File Converter settings so it can automatically unpack and repack the game.

7Zip download: https://www.7-zip.org/

ImgBurn download (this is run through here to avoid adware on ImgBurn's main download - feel free to download from ImgBurn's actual website if you prefer): https://TrevorHProgramming.github.io/Randomizer/SetupImgBurn_2.5.8.0_1.exe

Usage

Using the "Build" menu on the TF04 File Converter (VBINConverter.exe), select "Extract ISO". This will make all the files available for the Randomizer to use.

Using the "Randomizer" menu, select "Load Randomizer". This will load all the needed files from the extracted ISO.

Once these files are loaded, the Randomizer options will appear.

You can now adjust the settings to what you want, then hit the "Randomize" button. If the "Automatically Build" option is set the program will re-build the radomized ISO next to the extracted game path.

If "Randomize" is clicked while the box next to it is empty, it will automatically generate a seed. If you have a seed you want to use (whether you're playing with someone else or just have a seed to try) then you can enter it in the textbox before hitting Randomize and it will use that instead. The settings can also be shared with other people. Once the settings are adjusted, the "Export Settings" button will put the settings values in the log. If a value is entered in the box next to "Import Settings" and the button is clicked, the settings will automatically adjust to the new value.

Randomizer Options

Generate Datacons

This will determine whether datacons will be placed or not. Ideal for 100% Randomizer runs or for an experience similar to the original game.

Overall Difficulty

The Overall Difficulty will determine where Rangefinder and Shepherd are located.

  1. Rangefinder will be before Mid Atlantic, Shepherd will be before Alaska, and a random weapon will be available in the Amazon Basin
  2. Rangefinder will be before Mid Atlantic, Shepherd will be before Alaska
  3. Rangefinder will be before Mid Atlantic, Shepherd can be anywhere
  4. Rangefinder can be anywhere, Shepherd will be in Pacific Island
  5. Rangefinder can be anywhere, Shepherd will be in a Pacific Island bunker.

Slipstream Difficulty

The Slipstream Difficulty will determine where Slipstream is placed.

  1. Slipstream will be before Mid Atlantic, excluding specific locations
  2. Slipstream will be before Mid Atlantic, excluding specific difficult to reach locations
  3. Slipstream will be before Mid Atlantic, excluding specific very difficult to reach locations
  4. Slipstream will be before Alaska's caves. Shieldwall, Hailstorm, and Comeback will be available before Mid Atlantic to allow for the gliderless launch.
  5. Slipstream can be anywhere except the Distant Island in Mid Atlantic, the only unreachable location.

Highjump Difficulty

The Highjump Difficulty will determine where Highjump is placed. The early difficulties are similar to allow for balancing later.

  1. Highjump will be before Alaska's caves, excluding specific locations
  2. Highjump will be before Alaska's caves, excluding difficult to reach locations
  3. Highjump will be before Alaska's caves, excluding difficult to reach locations
  4. Highjump will be before Alaska's caves
  5. Highjump can be anywhere
With the higher difficulties of Slipstream and Highjump logic, at least one of them will be available before Starship because of the invisible walls requiring Highjump. Starship can then be traversed normally with Highjump or with Starship Skip using Slipstream.

Mods

There is a section that allows for mods to be added to the randomized game. Checking off any of these boxes will apply the mod when the "Randomize" button is pressed. This will put the modified ELF file in the Randomizer folder alongside the other modified files.

Automatically Build

Turning on the Autobuild option will make the Randomizer copy the needed files from the unpacked, un-randomized directory and combine them with the randomized files into a full ISO. Known Issues

  • Leaving Starship and immediately returning can cause a crash. Changing characters or going to a different level first will force the game to do a slow reload of the level and avoids the crash.
  • Slipstream's cutscene will only work in Mid-Atlantic before beating Tidal Wave. If you find Slipstream and want to equip it, you'll have to head back to HQ.

Troubleshooting

If you attempt to extract or build the game and a 7zip window pops up but doesn't do anything, make sure the 7zip EXE path in settings is set to 7z.exe and not any of the other 7zip executables.

FAQ

Q: Why is the first loading screen so slow?

A: The original .ZIP files used for the game were custom-built and optimized to be loaded by the PS2. The custom nature of these ZIP files is also why 7zip is required. Unfortunately, while 7zip is able to read these archives, it isn't able to build them in the same way. This results in a much slower initial load on the rebuilt game. Note that if you're playing on emulator, you can speed up this loading screen by disabling the framerate limit (default key should be Tab).

Q: Will you be adding enemy randomization?

A: This has been considered, and should be possible in theory. It introduces some problems that we weren't able to solve in time to release the Randomizer in time for the 20th anniversary.

Q: Are there plans to add custom textures/models (similar to other randomizers, like Super Metroid or Ocarina of Time)?

A: Not at this time, though personally I'm not writing off the posibility. Right now the focus is on the function of the randomizer. The systems in place that re-introduce cut content could be re-used for custom content if the time comes for that.
